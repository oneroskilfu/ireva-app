# iREVA Platform - Quick Start Guide

## Access the Application

1. Click the **Run** button at the top of the Replit interface
2. Wait for the application to load (10-15 seconds)
3. You should be automatically redirected to the main application
4. If not, click the "Click here if not redirected" button

## Alternative Access Methods

If the automatic redirection doesn't work, try:

1. **Direct access**: Use the URL `https://[replit-id]-5001.replit.dev/`
2. **Alternative URL formats**: Several URL options are provided on the loading page

## Troubleshooting

If you encounter issues:

1. Try refreshing the page
2. Use one of the direct port URLs from the loading page
3. Click Stop, then Run again to restart the application

For more detailed instructions, see APPLICATION-ACCESS-GUIDE.md
