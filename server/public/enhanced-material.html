Material Spacing

Material Design uses an 8dp square baseline grid for component dimensions, spacing, and touch targets.

All spacing in this document follows the 8dp grid (0.5rem), with measurements being multiples of this base unit.

Material Elevation

Material Design uses shadows to convey elevation. Elements closer to the user cast larger, softer shadows.

These cards demonstrate proper shadow elevation and hover states according to Material Design specifications.
